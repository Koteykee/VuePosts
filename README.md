# This simple posts app is my first Vue3 project where i learned how to:

1. Use axios to send requests to an API.
2. Use Pinia to store and synchronize the search state (searchQuery) across components.
3. Structure and reuse components in Vue3.
4. Use Suspense for handling asynchronous component loading.

## Installation
1. Clone the repo
```sh
git clone https://github.com/Koteykee/VuePosts
```
2. Install NPM packages
```sh
npm install
```
3. Run application
```sh
npm run dev
```
